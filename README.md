# CPP-DataStructure
**数据结构CPP实现**


## 线性表
+ 顺序表   seqList
+ 单链表   sLinkedList
+ 双向链表 dLinkedList

## 栈和队列
+ 顺序栈   seqStack
+ 链接栈   linkStack
+ 循环队列 seqQueue
+ 链接队列 linkQueue

## 树
+ 二叉树    binaryTree
+ 二叉搜索树 binarySearchTree
+ 平衡二叉树 avlTreee

## 堆
+ 最大堆、最小堆(优先级队列) minHeap、maxHeap、priorityQueue

## 哈希表   不同哈希算法
+ 开放寻址法 斐波那契散列+扰动算法处理冲突  openHashMap
+ 拉链法    链表长度大于8(红黑树dog)       linkHashMap

## 其他数据结构
+ 并查集  递归路径压缩 disJointSet
+ 字典树  Trie
+ 跳表  skipList
+ 布隆过滤器 bloomFilter

## 排序
+ 快速排序   插入+快排   quickSort
+ 归并排序   mergeSort
